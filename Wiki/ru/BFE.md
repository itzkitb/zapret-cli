# Управление службой Base Filtering Engine (BFE)

Служба **Base Filtering Engine (BFE)** критически важна для работы сетевого фильтрации Windows Defender Firewall. При её остановке могут возникнуть проблемы с сетевой безопасностью и подключением. Ниже приведены проверенные методы для запуска и восстановления службы.

> **Важно!** Все операции требуют прав **локального администратора**. Редактирование реестра выполняйте с осторожностью - некорректные изменения могут нарушить работу системы.

---

## Стандартный метод запуска
1. **Откройте консоль служб**  
   Нажмите `Win + R`, введите `services.msc`, подтвердите через **Enter**.

2. **Найдите службу**  
   В списке отыщите запись:  
   `Base Filtering Engine` (отображается как *«Базовая фильтрация»* в русской версии Windows).

3. **Проверьте статус**  
   • **Работает** — дополнительных действий не требуется.  
   • **Остановлена** — щелкните ПКМ → **Запустить**.  
   • **Отключена** — переходите к Альтернативному методу.

---

## Альтернативный метод: восстановление через реестр
> **Предупреждение**  
> Изменение системного реестра рекомендуется только при невозможности запуска через `services.msc`. Создайте точку восстановления системы перед началом.

1. Запустите **Командную строку от имени администратора**:
   ```cmd
   reg add "HKLM\SYSTEM\CurrentControlSet\services\BFE" /v Start /t REG_DWORD /d 2 /f
   ```
   > `2` означает автоматический запуск службы. Параметр `/f` принудительно применяет изменения.

2. Перезагрузите компьютер и проверьте статус службы через `services.msc`.

---

## Диагностика при постоянных сбоях
Если служба не запускается даже после указанных методов:

### 1. Проверка на вредоносное ПО
- Выполните полное сканирование с помощью проверенных решений:
  - **Kaspersky Virus Removal Tool** (бесплатная версия)
  - **Microsoft Safety Scanner** (официальный инструмент от Microsoft)

### 2. Восстановление системных файлов
В **административной командной строке** последовательно выполните:
```cmd
:: Проверка целостности системных файлов
sfc /scannow

:: Диагностика образа системы
DISM /Online /Cleanup-Image /CheckHealth
DISM /Online /Cleanup-Image /ScanHealth
DISM /Online /Cleanup-Image /RestoreHealth
```
> После завершения всех команд **обязательно перезагрузите ПК**.

---

## Крайние меры
Если проблема не решена:
1. **Обновите Windows** через «Центр обновления Windows».
2. **Проверьте конфликты ПО**:  
   Отключите сторонние антивирусы/брандмауэры через «Безопасный режим».
3. **Восстановите систему** до точки, где служба работала корректно.

> **Переустановка Windows** — финальный шаг при критическом повреждении системных компонентов. Перед этим сохраните пользовательские данные и создайте резервную копию раздел(а/ов).
